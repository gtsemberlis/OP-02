# Editorial Conventions

## Section Purpose

Canonical editorial conventions for OP-01.

## Canonical Content

### Editorial Voice

Write as though a highly competent investigator is documenting an operation for another highly competent investigator.

The voice is:

- professional;
- precise;
- restrained;
- observational;
- intellectually honest.

### Facts Before Interpretation

Present information in this order whenever practical:

1. observable facts;
2. professional assessment;
3. interpretation;
4. recommendations.

Do not imply certainty where none exists.

### Tense

- Use present tense for NPCs, locations, current operational state, and GM guidance.
- Use past tense for historical incidents, timelines, and recorded events.

### Preferred Terminology

| Preferred | Avoid |
|---|---|
| investigator | player |
| operation | adventure / scenario |
| phenomenon | creature / entity / organism |
| operational recommendation | ending |
| observation | revelation |

### Formatting

- Use prose for narrative.
- Use bullets for retrieval.
- Use tables for comparison.
- Use visual separators between major sections during rendering.
- Avoid decorative callouts, faux-classified graphics, and unnecessary visual effects.

### NPC and Institutional Standards

NPCs behave as complete individuals rather than plot devices. Institutions behave according to documented philosophies rather than narrative convenience.

### Clue Design

Evidence reinforces evidence. No single clue is indispensable. Important conclusions should be supported by multiple independent observations whenever practical.

### Revision Policy

Revisions are classified as:

- Editorial;
- Mechanical;
- Continuity;
- Major Revision.

All revisions originate in the Source Edition.
