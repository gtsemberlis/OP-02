# How to Use This Book

## Section Purpose

Canonical how to use this book for OP-01.

## Canonical Content

### Purpose

This Operation Book supports three phases:

- **Preparation:** understanding the operation before play;
- **Execution:** running the investigation at the table;
- **Review:** recording consequences and maintaining campaign continuity.

### Before the Session

Read the book in sequence the first time. Focus on:

- what has already happened;
- what each NPC knows;
- what each institution wants;
- how the evidence relates;
- how the world responds to investigator decisions.

Do not treat the operation as a scripted sequence.

### During the Session

Use this book as a reference, not a script. Ask:

- Who would know this?
- What evidence exists?
- How would this NPC respond?
- What is available at this location?
- What consequences follow this decision?

### After the Session

Record significant changes, including:

- NPC injuries or deaths;
- lost or recovered artifacts;
- changes to institutional relationships;
- investigator conclusions;
- outstanding questions;
- consequences that carry into later operations.

### Using the GM Console

The GM Console is the live operational dashboard. When additional context is needed, return to the Source Edition or Digital Reference Edition.

The Source Edition is authoritative. The GM Console is operational.

### Final Recommendation

Do not memorize the operation. Become familiar with its people, institutions, and relationships.

Run the world honestly. Trust the investigators. Allow the story to emerge.
