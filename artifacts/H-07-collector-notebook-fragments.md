# H-07 — Collector Notebook Fragments

## Artifact Type
Recovered field notebook / historical artifact

## Current Custody
Program evidence storage.

## Current Location
Recovered at L-02 — Alexandria Aviation Systems Headquarters; transferred to L-05.

## First Appearance
S-07 — The Collectors

## Description
A compact, heavily used handwritten notebook recovered from the deceased Collector.

## Presentation Notes
Paper, binding, and repairs suggest long service. The notebook contains no convenient exposition and no modern metadata.

## Investigator Access Conditions
Recovered only if investigators secure the deceased Collector or abandoned equipment.

## Canonical Content
The fragments contain:

- Biological sketches and symbolic diagrams.
- Measurements recorded in differing notational styles.
- References to locations outside Chicago.
- Observations apparently accumulated over decades.
- Cross-references whose source volumes are absent.
- No names, organization chart, mission orders, or electronic addresses.

## Operational Significance
H-07 demonstrates prior independent knowledge, institutional continuity, and a preservation culture fundamentally different from conventional espionage.

## Preservation Notes
Handle as both evidence and potentially hazardous anomalous research. The Program should preserve the original and create a complete facsimile.
